# CrewAI Hello World

## Installation

1. **Install Python:**
   - Download from [python.org/downloads](https://www.python.org/downloads/)
2. **Install uv:**
   - Follow [uv installation guide](https://docs.astral.sh/uv/getting-started/installation/)
3. **Install CrewAI Tools:**

   ```bash
   pip install 'crewai[tools]'

    # Check installation
   crewai --version

   ```

4. **Next Steps:**
